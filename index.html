  • 가는말이 고와야 오는말도 곱다. 주변사람, 업무적으로 만나는사람들에게는 최대한 친절하려고합니다.
  • 회사의 메뉴얼적인 업무를 못보고 지나칩니다. 어떻게하면 자동화 할 수 있을까 고민을 많이합니다.

Communication.

  • 원만한 대인관계를 지향합니다.
  • 하위 직급자에게 좋은 본보기가 되기 위해 노력합니다.
  • 내 업무 외에 타인의 업무를 같이 하려고 노력합니다.

Web.

  • 다양한 웹 시스템을 개발 및 유지보수 하였습니다.
  • 미흡하지만 Backend부터 Frontend까지 풀스택으로 가능합니다.
  • Frontend는 React를 선호하고, Backend는 Node를 선호합니다.

JavaScript.

  • 현재 저의 주로 사용하는 언어이고, 매우 재미있다고 생각하고 있습니다.
  • ES6 이후 문법을 선호하지만 레거시에서는 jQuery, ES5도 사용합니다.
  • 바닐라JS, Node 환경에서도 개발이 가능합니다.
  • 가끔 Opensource를 분석해서 이슈 수정도 합니다.

React.

  • 함수형 React 환경에서 주로 개발하고있습니다.
  • State관리에 중점을 두고있고, 상황에 맞는 랜더링 최적화도 함께 진행합니다.
  • 레거시 코드를 React 코드로 변환 추가 개발 가능합니다.
  • Jest를 통한 Unit test 작성이 가능합니다.

Node.

  • 사이드 프로젝트를 진행할때는 항상 Node Backend로 개발하였습니다.
  • Node Backend를 미흡하지만 간단하게 작성 가능합니다.
  • Electron 환경에서 개발을 하였습니다.

C#.

  • Javascript 이전에 저의 주 언어 였습니다.
  • 업무 뿐만아니라 사이드 프로젝트로 Window program을 개발할때 주로 사용합니다.
  • 제가 혼자 사용하는 메크로등도 제작에 능숙합니다.

Java.

  • C#과 비슷하여 사용하는데 크게 어려움이 없었습니다.
  • Sring을 사용할때 주로 사용했습니다.
